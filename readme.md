# Uber Clone Project

## Ongoing Project

## Overview
This project is a clone of the Uber app, providing similar functionalities such as ride booking, real-time tracking, and fare estimation.

## Features
- User Authentication
- Ride Booking
- Real-time GPS Tracking
- Fare Estimation
- Payment Integration

## Technologies Used
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Maps: Google Maps API
